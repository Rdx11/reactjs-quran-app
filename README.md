## Intallation
clone this repo `git clone https://github.com/Rdx11/reactjs-quran-app.git`
### install node modules and other packages :
```
yarn 
yarn add axios
yarn add react-loading-skeleton
yarn add react-router-dom
```
running this app `yarn start`
